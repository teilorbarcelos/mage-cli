# Playground

This directory is used to test the mage CLI during development.

Any files generated by the CLI will be created here. Feel free to delete everything inside `src/` at any time.

## Usage

From the project root:

```bash
# Run any mage command inside the playground
npm run play -- config show
npm run play -- patterns list
npm run play -- generate component UserCard
npm run play -- patterns init git@github.com:user/repo.git
```
